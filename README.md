# 🔐 Hash Analysis Toolkit

A Python-based Cybersecurity Toolkit designed for hash identification, hash generation, and security analysis through a modern graphical user interface (GUI).

## 📌 Features

* 🔍 Hash Identification
* 🔐 Hash Generation
* 📊 Security Analysis
* 🖥️ Modern Tkinter GUI
* 📁 Save Analysis Reports
* 📜 History Management
* 🎨 Dark Theme Interface
* 📈 Usage Statistics Counter

---

## 🛠️ Technologies Used

* Python 3
* Tkinter
* Colorama
* Hashlib
* Regular Expressions (re)

---

## 🔒 Supported Hash Types

| Hash Type | Security Level |
| --------- | -------------- |
| MD5       | Weak           |
| NTLM      | Weak           |
| LM        | Very Weak      |
| SHA1      | Weak           |
| SHA224    | Moderate       |
| SHA256    | Strong         |
| SHA384    | Strong         |
| SHA512    | Very Strong    |

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/rohinimate907/Hash-Analysis-Toolkit.git
```

Navigate to the project directory:

```bash
cd Hash-Analysis-Toolkit
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python gui.py
```
